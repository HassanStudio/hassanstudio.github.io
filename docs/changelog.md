## 获取更新

### 自动更新

Hassan 会定期检查更新，当有新版本可用时会通知你。

### 手动检查更新

1. 打开 Hassan
2. 点击菜单栏图标
3. 选择「关于 Hassan」
4. 点击「检查更新」按钮

### 下载最新版本

你也可以访问 Hassan 官网下载最新版本。
