# App 隐私政策

> 如有任何问题，请随时[联系我们](/about/contact)

Hassan 非常重视用户的隐私和数据安全。本隐私政策说明了 Hassan 如何处理用户信息。

## 简短声明

**Hassan 不收集、不上传、不分享任何用户数据。所有操作都在您的设备上本地完成。**

## 数据收集

### 我们不收集任何数据

Hassan 是一个完全本地化的应用程序，我们：

- ✅ **不收集**任何使用统计数据
- ✅ **不收集**任何崩溃报告
- ✅ **不收集**任何分析数据
- ✅ **不收集**任何个人信息
- ✅ **不收集**您的文件或脚本内容
- ✅ **不收集**您的系统信息

## 本地处理

Hassan 的所有功能都在您的设备上本地完成：

- **脚本执行**：在您的 Mac 上本地执行，不会发送到任何服务器
- **文件处理**：所有文件操作都在本地完成
- **配置管理**：所有设置都存储在您的 Mac 上
- **任务管理**：任务列表仅在本地扫描和管理

## 网络连接

**Hassan 不需要网络连接**：

- Hassan 不会主动建立任何网络连接
- Hassan 不会向任何服务器发送数据
- Hassan 不会检查更新（除非您手动触发）
- Hassan 可以在完全离线的环境中正常工作

## 权限使用

Hassan 可能需要以下系统权限才能正常工作：

### 1. 通知权限

**用途**：显示脚本执行结果的通知

**说明**：
- 用于告知您脚本执行成功或失败
- 通知内容仅包含执行结果，不包含敏感信息
- 通知仅在本地生成，不会发送到任何服务器
- 您可以在系统设置中随时关闭通知权限

### 2. 文件访问权限

**用途**：读取和处理您选择的文件

**说明**：
- 仅访问您在 Xcode 中明确选择要处理的文件
- 用于执行脚本时传递文件路径
- 不会自动访问您的其他文件
- 遵循 macOS 沙盒安全机制

### 3. 辅助功能权限（可选）

**用途**：某些高级脚本功能可能需要

**说明**：
- 仅在您的自定义脚本需要时才会请求
- 用于脚本执行系统级操作
- 可以随时在系统设置中撤销

## 数据存储

Hassan 在您的 Mac 上存储以下数据：

### 任务目录
```
~/Library/Application Support/Hassan/Tasks/
```
- 存储您的自定义脚本和内置脚本
- 完全在您的本地控制之下
- 可以随时添加、修改或删除

### 偏好设置
```
~/Library/Preferences/com.rakuyo.Hassan.plist
```
- 存储您的应用设置和配置
- 包括快捷任务、文件关联、解释器配置等
- 使用 macOS 标准的偏好设置存储机制

### 应用支持文件
```
~/Library/Application Support/Hassan/
```
- 存储应用运行所需的支持文件
- 包括内置脚本的备份等

**所有这些数据都存储在您的 Mac 上，不会同步到云端或发送到任何服务器。**

## 第三方服务

**Hassan 不使用任何第三方服务**：

- ❌ 无第三方分析服务（如 Google Analytics）
- ❌ 无崩溃报告服务（如 Crashlytics）
- ❌ 无广告服务
- ❌ 无云存储服务
- ❌ 无任何其他第三方服务

### 关于第三方代码库

虽然 Hassan 不使用任何第三方服务，但为了提供更好的开发体验和代码质量，Hassan 使用了一些优秀的开源代码库（Swift Package Manager 依赖）。

**重要说明**：
- 这些代码库都在您的设备上**本地运行**
- **不涉及任何数据收集**或网络传输
- 所有依赖库均为**开源项目**，代码公开透明可审查
- Hassan 遵守所有依赖库的开源许可证要求

主要使用的开源库包括：
- **The Composable Architecture (TCA)**：现代化的应用架构框架
- **SnapKit**：优雅的 Auto Layout 工具
- **RaLog**：强大的日志库
- 以及其他辅助工具库

完整的第三方开源库列表和许可证信息，请查看 [第三方开源许可证声明](/about/licenses)。

## 脚本执行安全

### 您的脚本

- 您的自定义脚本完全由您控制
- 脚本在您的 Mac 上以您的用户权限执行
- Hassan 不会修改、上传或分享您的脚本
- 脚本执行过程完全本地化

### 内置脚本

- Hassan 提供的内置脚本（复制路径、在 Finder 中显示等）
- 这些脚本的源代码是公开透明的
- 仅执行简单的本地操作（复制到剪贴板、打开 Finder 等）
- 不会访问网络或执行危险操作

## 数据安全

Hassan 采取以下措施保护您的数据安全：

1. **本地处理**：所有数据处理都在本地完成，不会离开您的设备
2. **沙盒隔离**：遵循 macOS 沙盒机制，保护系统安全
3. **权限最小化**：仅请求必要的权限
4. **无网络连接**：不建立任何网络连接，杜绝数据泄露风险
5. **开源透明**：文档开源，代码行为透明可查

## 用户权利

作为 Hassan 的用户，您拥有以下权利：

### 1. 数据访问权
您可以随时访问 Hassan 存储的所有数据：
- 任务脚本位于 `~/Library/Application Support/Hassan/Tasks/`
- 配置文件位于 `~/Library/Preferences/com.rakuyo.Hassan.plist`

### 2. 数据删除权
您可以随时删除所有数据：
```bash
# 删除任务目录
rm -rf ~/Library/Application\ Support/Hassan

# 删除偏好设置
defaults delete com.rakuyo.Hassan
```

### 3. 权限撤销权
您可以在 macOS 系统设置中随时撤销授予 Hassan 的权限：
- 系统设置 → 隐私与安全性 → 通知
- 系统设置 → 隐私与安全性 → 文件和文件夹

### 4. 完全控制权
- 您完全控制 Hassan 访问的所有数据
- 您可以随时查看、修改或删除任何数据
- 您可以随时卸载 Hassan

- 在应用中显示通知

## 合规声明

Hassan 遵守以下隐私法规：

- **GDPR**（欧盟通用数据保护条例）：由于我们不收集任何数据，Hassan 天然符合 GDPR 要求
- **CCPA**（加州消费者隐私法案）：我们不出售或分享任何个人信息
- **Apple App Store 隐私要求**：Hassan 完全符合 Apple 的隐私指南

## 总结

Hassan 的隐私承诺：

- ✅ **零数据收集** - 我们不收集任何数据
- ✅ **本地处理** - 所有操作都在您的设备上完成
- ✅ **无网络连接** - 不需要网络，不会上传数据
- ✅ **完全透明** - 文档开源，行为透明
- ✅ **用户控制** - 您完全控制所有数据

您的隐私和数据安全是我们的首要任务。
