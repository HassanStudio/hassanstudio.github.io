# 关于 <!--@include: @/.vitepress/snippets/app-name.md-->

欢迎使用 <!--@include: @/.vitepress/snippets/app-name.md-->！

## 什么是 <!--@include: @/.vitepress/snippets/app-name.md-->

<!--@include: @/.vitepress/snippets/app-name.md--> 是为 Xcode 量身打造的脚本执行器，灵感来源于《Fate》中 “山中老人”（Hassan-i-Sabbah）。它如同暗影中的刺客，隐匿无形，却以敏捷身手解决开发中的痛点。

## 为什么开发 <!--@include: @/.vitepress/snippets/app-name.md-->

开发的初衷是因为 Xcode 没有提供像是 VS Code/Cursor 那样的 “复制相对路径” 功能，而我经常在使用各类 CLI AI 工具时使用到这个功能。

一次偶然的机会，Xcode 中一个不起眼的功能 “Open with External Editor” 引起了我的注意，并以此为突破口开发出更通用的脚本执行器 <!--@include: @/.vitepress/snippets/app-name.md-->

## 核心设计理念

- **简单易用** - 无需复杂配置，开箱即用
- **高度可定制** - 支持自定义脚本，满足各种自动化需求
- **性能优先** - 轻量级设计，不影响开发效率
- **隐私至上** - 所有操作本地完成，不收集任何数据

## 隐私与安全

<!--@include: @/.vitepress/snippets/app-name.md--> 非常重视你的隐私和数据安全：

- [x] **本地处理**：所有操作都在本地完成，不会上传任何数据
- [x] **无数据收集**：不收集任何使用统计或分析数据
- [x] **无网络连接**：不需要网络连接即可正常工作
- [x] **沙盒隔离**：遵循 macOS 沙盒机制，保护系统安全
- [x] **权限透明**：只请求必要的权限，并明确说明用途

详细信息请查看[隐私政策](/zh/about/privacy)。

## 系统要求

- **操作系统** - macOS 15.0 (Sequoia) 或更高版本
- **开发工具** - 任意版本的 Xcode
