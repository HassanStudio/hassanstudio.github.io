# 快速开始

本指南将帮助你快速安装和配置 Hassan，让你在几分钟内开始使用。

## 系统要求

在开始之前，请确保你的系统满足以下要求：

- **操作系统**: macOS 15.0 (Sequoia) 或更高版本

::: info 说明
Hassan 专为 Xcode 设计，需要与 Xcode 配合使用才能发挥作用。支持任意版本的 Xcode。
:::

**App Store 链接**：[Hassan on App Store](https://apps.apple.com/app/id6755989817)

## 授予必要权限

Hassan 可能需要以下权限才能正常工作：

- **通知权限** - 用于显示脚本执行结果的通知
- **文件访问权限** - 用于读取和处理文件

系统会在需要时自动弹出权限请求，请根据提示授予权限。

## 配置 Hassan

### 配置文件关联

要在 Xcode 中使用 Hassan，需要将 Hassan 设置为某些文件类型的默认编辑器。

::: warning 重要提示
配置文件关联会修改这些文件类型的默认应用程序。Hassan 会记住原始设置，你可以随时通过「恢复默认」来改回去。
:::

**通过 Hassan 设置窗口配置：**

1. 启动 Hassan
2. 使用快捷键 `Command + ,` 或通过菜单栏点击「偏好设置...」打开设置窗口
3. 切换到「文件关联」标签页
4. 在这里你可以：
   - **新增关联**：点击 `+` 按钮添加自定义文件扩展名
   - **删除关联**：选中文件类型后点击 `-` 按钮
   - **设置关联**：勾选你想要关联的文件扩展名（如 `.swift`、`.h`、`.m` 等）
   - **应用设置**：点击「应用设置」按钮使更改生效
   - **恢复默认**：点击「恢复默认」来撤销所有关联更改

::: tip 提示
建议先选择几个常用的文件类型进行关联，比如 `.swift`。你可以随时回来添加或删除文件类型。
:::

### 配置快捷任务（可选）

快捷任务允许你设置一个默认脚本，在打开文件时自动执行，无需每次选择：

1. 打开设置窗口（`Command + ,`）
2. 在「通用」标签页中找到「快捷任务」设置
3. 勾选「启用」复选框
4. 从下拉菜单中选择要执行的脚本（如「复制相对路径」）
5. （可选）勾选「快捷模式」，这样会直接执行脚本，不显示任务选择菜单

::: warning 注意
启用快捷模式后，通过「在外部编辑器中打开」时会直接执行脚本，不再显示菜单。如果需要执行其他脚本，请通过菜单栏访问。
:::

## 基本使用

配置完成后，你可以开始在 Xcode 中使用 Hassan 了：

### 在 Xcode 中使用

1. 在 Xcode Project Navigator 中找到任意文件
2. 右键点击文件
3. 选择「**Open with External Editor**」（或「在外部编辑器中打开」）
4. Hassan 会显示任务列表（或直接执行快捷任务）
5. 选择你想要执行的脚本

### 通过菜单栏使用

Hassan 在菜单栏提供了图标（状态栏），方便你快速访问各项功能。

::: tip 提示
如果你没看到菜单栏图标，请在设置窗口的「通用」标签页中勾选「显示菜单栏图标」。
:::

点击菜单栏中的 Hassan 图标（🔨），你可以：

**1. 快捷任务管理**
- **启用快捷任务** - 开关快捷任务功能
- **快捷任务** - 子菜单，选择默认执行的脚本

**2. 文件关联管理**
- **文件关联** - 子菜单，快速勾选/取消文件类型关联
  - 直接点击文件扩展名（如 `.swift`）即可切换关联状态
  - 点击「配置扩展名...」可打开设置窗口的文件关联页面

**3. 快速访问**
- **偏好设置...** - 打开设置窗口（快捷键 `Command + ,`）
- **退出 Hassan** - 退出应用（快捷键 `Command + Q`）

::: tip 提示
通过菜单栏可以快速管理快捷任务和文件关联，无需每次都打开设置窗口。
:::

## 验证安装

让我们验证 Hassan 是否正常工作：

### 测试复制相对路径

1. 在 Xcode 中打开任意项目
2. 在 Project Navigator 中右键点击任意文件
3. 选择「Open with External Editor」
4. 选择「复制相对路径」脚本
5. 打开任意文本编辑器，使用 `Command + V` 粘贴

如果看到文件的相对路径被粘贴出来，说明 Hassan 已经正常工作了！

## 首选项设置

Hassan 提供了丰富的自定义选项：

### 通用设置

- **外观模式**: 浅色 / 深色 / 跟随系统
- **语言**: 中文 / English / 跟随系统
- **后台运行**: 关闭窗口时保持应用在后台运行
- **显示菜单栏图标**: 在系统菜单栏显示 Hassan 图标
- **快捷任务**: 配置默认脚本和快捷模式

### 任务管理

- 查看所有可用脚本的树形列表
- 在 Finder 中打开任务目录
- 管理自定义脚本

### 解释器配置

- 自动检测系统可用的脚本解释器
- 调整解释器优先级
- 配置搜索路径和命令名

### 文件关联

- 批量设置文件类型关联
- 查看当前关联状态
- 一键恢复默认设置
