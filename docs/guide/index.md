# 介绍

欢迎使用 Hassan！Hassan 是一个专为 macOS 开发者设计的智能脚本执行平台，它伪装成外部编辑器，为你的开发工作流带来强大的自动化能力。

## 什么是 Hassan？

Hassan 是一个智能的 macOS 脚本执行平台，灵感来自《Fate》系列中的"山中老人"（Hassan-i-Sabbah）。它的核心理念是"隐藏在明处" —— 伪装成普通的外部编辑器，实际上提供了强大的脚本执行和自动化功能。

## 为什么选择 Hassan？

在日常开发中，我们经常需要对文件执行各种操作：
- 复制文件路径用于配置或文档
- 在 Finder 中快速定位文件
- 对文件执行自定义的处理脚本
- 批量处理特定类型的文件

这些操作通常需要在多个应用之间切换，或者记住复杂的快捷键。Hassan 通过伪装成外部编辑器的方式，让你可以在 Xcode 中通过简单的右键菜单完成所有这些操作。

## 核心理念

### "隐藏在明处"

Hassan 不会以独立应用的形式干扰你的工作流程，而是巧妙地集成到 Xcode 的"在外部编辑器中打开"功能中。当你在 Xcode 中右键点击文件并选择"Open with External Editor"时，Hassan 会接管这个操作，让你可以：

1. **执行脚本** - 选择任意脚本对文件进行处理
2. **快捷模式** - 配置默认脚本，直接执行无需选择
3. **菜单访问** - 通过菜单栏快速访问所有功能

### 灵活的脚本系统

Hassan 支持多种脚本语言（Shell、Python、Ruby、Swift 等），你可以：

- 使用内置的实用脚本
- 编写自己的自动化脚本
- 组织脚本到层级目录
- 通过元数据为脚本添加标题和描述

## 主要特性

### 🎭 智能伪装
伪装成外部编辑器，注册到 macOS 系统中，无缝集成到 Xcode 工作流。

### ⚡️ 脚本执行引擎
- 支持多种脚本语言（Shell、Python、Ruby、Swift 等）
- 智能解释器匹配，自动选择合适的执行环境
- 传递文件路径作为参数，方便脚本处理

### 📋 任务管理系统
- 自动扫描 `~/Library/Application Support/Hassan/Tasks/` 目录
- 支持层级分组和树形展示
- 智能过滤（隐藏文件、临时文件等）
- 脚本元数据标注（@title、@description、@author 等）

### 🔗 文件关联管理
- 批量设置文件类型的默认编辑器
- 支持自定义文件扩展名
- 一键恢复到原始默认编辑器
- 保存原始关联状态，方便回滚

### 🎯 快捷任务
- 配置默认脚本，自动执行无需选择
- 快捷模式直接执行，不显示菜单
- 提升高频操作的效率

### 📦 内置脚本
Hassan 提供以下内置脚本，开箱即用：

1. **复制绝对路径** - 复制文件的绝对路径到剪贴板
2. **复制相对路径** - 智能计算项目根目录并复制相对路径
   - 支持 `.xcworkspace`（monorepo）
   - 支持 `Tuist/` + `mise.toml`（Tuist 项目）
   - 支持 `.git`（Git 仓库）
   - 支持 `.xcodeproj`（Xcode 项目）
3. **在 Finder 中显示** - 在 Finder 中定位并选中文件

## 适用场景

Hassan 特别适合以下场景：

### 场景 1：复制文件路径
在编写配置文件、文档或构建脚本时，需要引用其他文件的路径。

### 场景 2：文件定位
快速在 Finder 中找到文件，进行重命名、移动等操作。

### 场景 3：自动化处理
对特定文件执行自定义的自动化脚本，比如代码生成、格式转换等。

### 场景 4：批量操作
通过文件关联功能，让 Hassan 成为特定文件类型的默认处理程序。

## 系统要求

- **操作系统**: macOS 15.0 (Sequoia) 或更高版本
- **开发工具**: Xcode 14.0 或更高版本（如果需要使用 Xcode 集成功能）
