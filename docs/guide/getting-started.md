# 快速开始

本指南将帮助你快速安装和配置 Hassan，让你在几分钟内开始使用。

## 系统要求

在开始之前，请确保你的系统满足以下要求：

- **操作系统**: macOS 15.0 (Sequoia) 或更高版本
- **开发工具**: Xcode 14.0 或更高版本（如果需要使用 Xcode 集成功能）

::: tip 提示
虽然 Hassan 主要设计用于 Xcode 集成，但也可以作为独立应用使用，通过菜单栏访问所有功能。
:::

## 下载与安装

### 方式 1：从官网下载（推荐）

访问 Hassan 官网下载最新版本：
1. 下载 `Hassan.app.zip`
2. 解压下载的文件
3. 将 `Hassan.app` 拖拽到「应用程序」文件夹

### 方式 2：通过 Homebrew 安装

```bash
# 即将支持
# brew install --cask hassan
```

## 首次启动

### 1. 打开 Hassan

首次启动 Hassan 时，macOS 可能会显示安全警告。请按照以下步骤操作：

1. 在「应用程序」文件夹中找到 Hassan.app
2. 按住 `Control` 键并点击应用图标
3. 在弹出菜单中选择「打开」
4. 在安全对话框中点击「打开」

::: info 说明
这是 macOS Gatekeeper 的安全保护机制。完成首次打开后，之后就可以正常双击启动了。
:::

### 2. 授予必要权限

Hassan 可能需要以下权限才能正常工作：

- **通知权限** - 用于显示脚本执行结果的通知
- **文件访问权限** - 用于读取和处理文件

系统会在需要时自动弹出权限请求，请根据提示授予权限。

## 配置 Hassan

### 配置文件关联

要在 Xcode 中使用 Hassan，需要将 Hassan 设置为某些文件类型的默认编辑器。Hassan 提供了两种配置方式：

#### 方式 1：通过 Hassan 设置窗口（推荐）

这是最简单的方法，Hassan 会帮你批量设置文件关联：

1. 启动 Hassan
2. 使用快捷键 `Command + ,` 或通过菜单栏点击「偏好设置...」打开设置窗口
3. 切换到「文件关联」标签页
4. 勾选你想要关联的文件扩展名（如 `.swift`、`.h`、`.m` 等）
5. 点击「应用设置」按钮

Hassan 会记住原始的文件关联设置，你可以随时点击「恢复默认」来撤销更改。

::: tip 提示
建议先选择几个常用的文件类型进行关联，比如 `.swift`。你可以随时回来添加更多文件类型。
:::

#### 方式 2：通过 Finder 手动设置

如果你只想为特定文件类型设置关联，可以使用 Finder：

1. 在 Finder 中找到任意目标文件（如 `.swift` 文件）
2. 右键点击文件，选择「显示简介」（或使用快捷键 `Command + I`）
3. 在简介窗口中，展开「打开方式」部分
4. 从下拉菜单中选择 Hassan.app
5. 点击「全部更改...」按钮
6. 在确认对话框中点击「继续」

这样，所有相同类型的文件都会使用 Hassan 打开。

### 配置快捷任务（可选）

快捷任务允许你设置一个默认脚本，在打开文件时自动执行，无需每次选择：

1. 打开设置窗口（`Command + ,`）
2. 在「通用」标签页中找到「快捷任务」设置
3. 勾选「启用」复选框
4. 从下拉菜单中选择要执行的脚本（如「复制相对路径」）
5. （可选）勾选「快捷模式」，这样会直接执行脚本，不显示任务选择菜单

::: warning 注意
启用快捷模式后，通过「在外部编辑器中打开」时会直接执行脚本，不再显示菜单。如果需要执行其他脚本，请通过菜单栏访问。
:::

## 基本使用

配置完成后，你可以开始在 Xcode 中使用 Hassan 了：

### 在 Xcode 中使用

1. 在 Xcode Project Navigator 中找到任意文件
2. 右键点击文件
3. 选择「**Open with External Editor**」（或「在外部编辑器中打开」）
4. Hassan 会显示任务列表（或直接执行快捷任务）
5. 选择你想要执行的脚本

### 通过菜单栏使用

Hassan 也提供了菜单栏图标，方便你快速访问：

1. 点击菜单栏中的 Hassan 图标
2. 从菜单中选择「任务」
3. 选择你想要执行的脚本
4. 在文件选择器中选择要处理的文件

## 验证安装

让我们验证 Hassan 是否正常工作：

### 测试复制相对路径

1. 在 Xcode 中打开任意项目
2. 在 Project Navigator 中右键点击任意文件
3. 选择「Open with External Editor」
4. 选择「复制相对路径」脚本
5. 打开任意文本编辑器，使用 `Command + V` 粘贴

如果看到文件的相对路径被粘贴出来，说明 Hassan 已经正常工作了！

### 测试在 Finder 中显示

1. 在 Xcode 中右键点击任意文件
2. 选择「Open with External Editor」
3. 选择「在 Finder 中显示」脚本
4. Finder 应该会打开并选中该文件

## 首选项设置

Hassan 提供了丰富的自定义选项：

### 通用设置

- **外观模式**: 浅色 / 深色 / 跟随系统
- **语言**: 中文 / English / 跟随系统
- **后台运行**: 关闭窗口时保持应用在后台运行
- **显示菜单栏图标**: 在系统菜单栏显示 Hassan 图标
- **快捷任务**: 配置默认脚本和快捷模式

### 任务管理

- 查看所有可用脚本的树形列表
- 在 Finder 中打开任务目录
- 管理自定义脚本

### 解释器配置

- 自动检测系统可用的脚本解释器
- 调整解释器优先级
- 配置搜索路径和命令名

### 文件关联

- 批量设置文件类型关联
- 查看当前关联状态
- 一键恢复默认设置
